## 安裝說明
command terminal輸入
/*
wtf
*/
下載npm10.9.2(最新)跟python版本3.X.X
下載整個專案檔案，放在一個資料夾裡，資料夾命名navigator-demo
navigator-demo目錄下新建.env檔案，.env檔範例看.env.example，.env裡面放你的api(目前是用groq因為免費)，去這個網站拿api:https://console.groq.com/home
開commnad terminal，進到navigator-demo資料夾底下，輸入npm start
瀏覽器打開連結:http://localhost:3000/

## 版本控制
每次改東西之前： git pull #拉最新的版本出來
改完之後：
        git add .
        git commit -m "新增了什麼功能，屬名"
        git push
